## [Окружение](declarations.d.ts)
- `DEV`: Запуск в режиме long-pooling
- `HOST`: Хост для вебхука
- `PORT`: Порт
- `SSL_PRIVATE_KEY`: SSL приватный ключ
- `SSL_CERTIFICATE`: SSL сертификат
- `WHITELIST_IDS`: Разрешенные id пользователей
- `BOT_TOKEN`: Токен бота
- `SECRET_TOKEN`: Секретный токен вебхука
- `MONGO_URL`: URL MongoDB
- `CA_CHAT_ID`: Чат центра сертификации для публикации сертификатов
- `CA_TOPIC_ID`: Топик в чате CA
- `PUBLISH_DESTINATIONS`: Каналы для публикации в формате `alias:chatId?:topicId`
- `START_MESSAGE`: Текст сообщения на команду `/start`
- `SIGN_MODE`: `text-plain` - подпись текста сообщения, `text-markup` - подпись текста и разметки

## Команды

### Чат с ботом
- `/set_pubkey`: Установить сертификат
- `/revoke_pubkey`: Отозвать сертификат
- `/verify`: Проверить подпись / завершить проверку подписи
- `/alias`: Показать алиасы каналов для публикации
- `/scene`: Текущая сцена
- `/leave`: Выйти из текущей сцены

### Групповой чат
Команды используются с ответом на сообщение
- `/sign`: Подписать сообщение
- `/revoke_sign`: Отозвать подпись сообщения
- `/publish`: Опубликовать подписанное сообщение
- `/state_sigs`: Состояние подписей сообщения
- `/get_sign_data`: Получить файл сообщения для подписи
